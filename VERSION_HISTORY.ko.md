# 버전 이력

이 문서는 커밋 히스토리를 기반으로 Semantic Versioning(MAJOR.MINOR.PATCH)을 따르는 버전 업데이트 요약입니다.

## 1.0.0

- 추가: 회고, 목표 설정, 액션플랜, 리포트까지 이어지는 전체 제품 플로우
- 추가: AI 추천 및 리포트 파이프라인(커스텀 프롬프트, 피드백 포함)
- 추가: 리포트 및 그리드 PDF 출력
- 추가: 크로스 플랫폼 로그인 코드 기반 인증 개선
- 추가: 관리자 대시보드 기반 및 진행 가이드
- 변경: UI 레이아웃 개선 및 프리뷰 개선
- 변경: Gemini 설정 변경 및 모델 디프리케이션 대응
- 수정: 로그인 콜백 오류 및 검증 엣지 케이스
- 수정: 화이트리스트 및 API 키 처리 등 보안 업데이트
